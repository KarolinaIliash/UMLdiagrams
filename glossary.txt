Storage - назва класу, який узагальнює всю систему загалом
finishOrder() - функція, що включає в себе прибирання замовлення з списку незавершених, а також редагування інформації про товари
Item - клас товару
minAmount - мінімальна кількість товару. Якщо менше - потрібно докупляти.
storagePlace - номер полички на складі, де зберігається товар
Position - клас позиції для замовлення, що містить назву товару та його кількість в замовленні
Order - клас замовлення, що містить список позицій, id та постачальника, що має реалізувати дане замовлення
Provider(class) - клас для нашого розуміння постачальника
unfinishedOrders - список замовлень, які даний постачальник ще не реалізував
finishOrder() - функція, що прибирає замовлення зі списку незавершених замовлень


Admin-узагальнена назва для працівника лавки
Provider(actor) - постачальник товару
Make Order - загальна назва для замовлення товару 
Make official agreement - укладання договору
Accept Order - прийняття замовлення постачальнтком
Send Order - надсилання замовлення постачальником
Get Order - отримання замовлення адміном


Sufficient quantity of items - достатня кількість товарів на складі, не потрібно нічого докуплтяти
Sorting out items - розкладання товару по поличках після його отримання
Checking availability of new items - чи є в прийнятих товарах товари, яких ше немає


Assign order its id - присвоїти замовленню унікальне id
